ПРИМЕЧАНИЕ

Для использования бота на вашем компьютере должен быть установлен Python 3.7.x . Но для использования самой программы Python 3.7.x не нужен! Для запуска бота устанавливать какие-либо библиотеки НЕ НУЖНО. Бот всё сделает сам при запуске.

ОПИСАНИЕ

В этом репозитории находится генератор наипростейших (наитупейших) ботов ВКонтакте.
Это приложение способно создавать ботов, умеющих отвечать на сообщение от пользователей.

УСТАНОВКА

1. Создайте свою группу, включите в ней сообщения
2. Перейдите в на главную страницу вашего сообщества (требуются права администратора), далее перейдите в <Управление -> Работа с API>
2.1 Во вкладке <Ключи доступа> создайте ключ доступа, активировав все галочки, и поместите его в соответствующий пункт в программе.
2.2 Перейдите во вкладку Long Poll API и включите Long Poll API. Установите новейшую версию Long Poll API и во вкладке <Типы событий> активируйте все галочки (для ботов, созданных с помощью данной программы, хватит только прав во вкладке <Сообщения>)
3. Теперь скопируйте ID вашей группы (по стандарту идёт после слова club в ссылке на сообщество)
4. Вводим название бота (желательно, без формата)
5. Добавляем все нужные нам команды. Пока что бот умеет только отвечать на заданные сообщения. В будущем его функционал, возможно, расширится.
6. Нажимаем кнопку <Сгенерировать бота>. Python файл (формат .py) появится на рабочем столе. Если данная программа будет интересна, то в будущем может появиться возможность выбирать директорию для сохранения файла бота.
7. Запускаем нашего бота (см. ПРИМЕЧАНИЕ).
